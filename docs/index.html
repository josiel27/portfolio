Sistema para controle de tarefas, desenvolvido na linguagem de programação JavaScript com as ferramentas Ionic e Firebase para guardar os dados da aplicação. As tecnologias usadas foram: AngularJS, Jquery, Ionic, Firebase, Bootstrap, Font Awesome, CSS3, HTML5.  Esse  sistema permite a inserção, alteração, exclução e a visualização de tarefas com o controle da data de entrega da tarefa e uma descrição.
